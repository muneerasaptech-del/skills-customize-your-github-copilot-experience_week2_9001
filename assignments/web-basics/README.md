# 📘 Assignment: Web Basics - HTML & CSS

## 🎯 Objective

Learn the fundamentals of web development by creating a personal portfolio webpage using HTML and CSS. You'll practice structuring content with HTML and styling it with CSS to create a visually appealing web page.

## 📝 Tasks

### 🛠️ Build Page Structure with HTML

#### Description
Create a well-structured HTML page that includes a header, navigation, main content sections, and a footer for your personal portfolio.

#### Requirements
Completed program should:

- Create a proper HTML5 document structure with DOCTYPE, html, head, and body tags
- Include a header with your name and a tagline
- Add a navigation bar with links to different sections (Home, About, Projects, Contact)
- Create at least three main content sections (About, Projects, Contact) with meaningful content
- Include a footer with copyright information
- Use semantic HTML elements (header, nav, main, section, footer)


### 🛠️ Style with CSS

#### Description
Apply CSS styling to make your portfolio visually appealing and improve the user experience with colors, fonts, spacing, and layout.

#### Requirements
Completed program should:

- Style the header and navigation bar with background colors and proper spacing
- Apply a cohesive color scheme throughout the page
- Use different font families and sizes for visual hierarchy
- Add padding and margins to create proper spacing between elements
- Style links with hover effects
- Make the layout responsive or centered on the page
- Ensure all text is readable with good contrast between text and background
